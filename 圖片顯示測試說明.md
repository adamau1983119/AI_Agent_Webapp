## 🔍 前端顯示方式確認

### ImageGallery 組件（主題詳情頁）
```tsx
<img
  src={image.url}  // ✅ 使用 <img> 標籤顯示實際圖片
  alt={`Image ${image.order}`}
  className="w-full h-full object-cover"
  onError={(e) => {
    e.currentTarget.src = 'https://via.placeholder.com/400x400?text=Image'
  }}
/>
```

### ImageSearch 組件（搜尋結果）
```tsx
<img
  src={image.url}  // ✅ 使用 <img> 標籤顯示實際圖片
  alt={image.id}
  className="w-full h-full object-cover"
  onError={(e) => {
    e.currentTarget.src = 'https://via.placeholder.com/400x400?text=Image'
  }}
/>
```

**結論：** ✅ 前端使用 `<img src={url}>` 顯示實際圖片，不是只顯示 URL 文字
